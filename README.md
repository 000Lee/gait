# 하지·보행 인수인계 관찰표

**동작 관찰 → 약한 근육(감별) → 인수인계 표현**

재활병원 환자(편마비·척수손상·전신약화·정형 혼합) 하지·보행 중심.
"약한 근육"은 **확정이 아니라 감별 대상**이다. 근약화와 경직은 원인이 정반대일 수 있으니, 확신이 없으면 "약화 의심 / 경직 감별 필요"로 여지를 남기는 표현이 더 정확하고 방어 가능하다. MMT 등급을 확인했으면 수치로 못 박는다.

---

## 1. 기립·착석 (Sit-to-Stand / Sit-down)

| 관찰되는 동작/현상 | 약한 근육·기능 (감별) | 인수인계 표현 예시 |
|---|---|---|
| 일어설 때 팔로 밀고 엉덩이가 잘 안 떨어짐 | 대퇴사두근, 대둔근 (하지 신전근군) | "하지 신전근군 약화로 STS 시 상지 의존, Min~Mod A 필요" |
| 앉을 때 툭 떨어지듯 주저앉음 | 대퇴사두근 (원심성 조절) | "Quadriceps 원심성 조절 저하로 controlled lowering 어려움" |
| 일어설 때 체간을 충분히 앞으로 못 숙임 | 체간 굴곡·고관절 굴곡 협응 | "STS 시 anterior weight shift 부족으로 기립 개시 지연" |
| 일어선 직후 휘청이며 균형 못 잡음 | 고관절 신전·외전근 + 균형 반응 | "기립 직후 정적 균형 저하, 초기 수 초간 근접 감독 필요" |
| 한쪽 다리로만 밀고 일어남 | 편측 하지 신전근군 (반대측 약화) | "편측 하지 의존적 STS, 환측 loading 감소" |

---

## 2. 서 있는 자세 (Standing / Static Stance)

| 관찰되는 동작/현상 | 약한 근육·기능 (감별) | 인수인계 표현 예시 |
|---|---|---|
| 무릎이 뒤로 꺾임(과신전) | 대퇴사두근 약화 **또는** 족저굴곡근 경직 | "Stance 시 knee hyperextension, quad 약화/PF 경직 감별 필요" |
| 한 다리로 설 때 골반이 반대쪽으로 떨어짐 | 중둔근 (고관절 외전근) | "Trendelenburg sign (+), 편측 지지 시 골반 안정성 저하" |
| 체중을 한쪽(건측)으로만 실음 | 환측 하지 근력·감각·자신감 저하 | "Weight bearing asymmetry, 환측 loading 회피" |
| 오래 서 있으면 점점 무너짐 | 근지구력 + 체간 안정성 | "정적 기립 지구력 저하로 시간 경과 시 자세 붕괴" |
| 발이 바깥/안으로 돌아간 채로 섬 | 고관절 회전근 불균형 / 경직 | "Stance 시 하지 정렬 이상, 회전근 불균형 관찰" |

---

## 3. 유각기 문제 (Swing Phase / Foot Clearance)

| 관찰되는 동작/현상 | 약한 근육·기능 (감별) | 인수인계 표현 예시 |
|---|---|---|
| 발끝이 바닥에 끌림 / 발이 안 들림 | 전경골근 (족배굴곡근) → foot drop | "우측 foot drop, tibialis anterior 약화, swing 시 toe drag" |
| 발을 반원 그리듯 돌려서 내딤 | 족배굴곡·슬관절굴곡 약화 보상 | "Foot clearance 부족으로 circumduction gait" |
| 골반 한쪽을 들어올려 다리를 빼냄 | clearance 확보 위한 보상 | "Swing 시 hip hiking으로 clearance 확보" |
| 다리를 앞으로 못 내밈 / 끌고 나옴 | 장요근 (고관절 굴곡근) | "Swing initiation 저하, hip flexor 약화로 stepping 어려움" |
| 무릎을 못 굽혀 다리가 뻣뻣하게 나옴 | 슬관절 굴곡근 약화 **또는** 신전근 경직 (stiff-knee) | "Swing 시 knee flexion 부족(stiff-knee), 약화/경직 감별 필요" |

---

## 4. 입각기 문제 (Stance Phase / Loading)

| 관찰되는 동작/현상 | 약한 근육·기능 (감별) | 인수인계 표현 예시 |
|---|---|---|
| 디딜 때 무릎이 확 꺾임(buckling) | 대퇴사두근 | "Loading response에서 knee buckling, quadriceps 약화 의심" |
| 발뒤꿈치가 안 닿고 앞꿈치부터 딛음 | 족저굴곡근 경직 **또는** 족배굴곡근 약화 | "Heel strike 소실, forefoot contact, PF 경직/DF 약화 감별" |
| 밀고 나가는 힘이 없어 보폭이 짧음 | 비복근·가자미근 (push-off) | "Push-off 약화로 stride length 감소" |
| 입각기에 체간이 디딘 다리 쪽으로 기욺 | 중둔근 (보상성 체간 편위) | "보상성 lateral trunk lean, 중둔근 약화 소견" |
| 발바닥 전체가 철퍽 떨어짐(foot slap) | 전경골근 원심성 조절 저하 | "Loading 시 foot slap, tibialis anterior 원심성 조절 저하" |

---

## 5. 보행 전반·지구력 (Gait General / Endurance)

| 관찰되는 동작/현상 | 약한 근육·기능 (감별) | 인수인계 표현 예시 |
|---|---|---|
| 보폭이 좌우 비대칭 | 편측 하지 근력·경직 불균형 | "Step length asymmetry, 환측 단축" |
| 걸을수록 편위가 심해지고 무너짐 | 전반적 근지구력 + 체간 안정성 | "지구력 저하로 보행 후반부 편위 심화, 거리 제한적" |
| 계단·경사 오를 때 유독 힘들어함 | 고관절 신전·외전근, 대퇴사두근 | "계단 상행 시 하지 근력 부족, 난간 의존 및 감독 필요" |
| 계단 내려갈 때 더 불안·주저 | 대퇴사두근 원심성 조절 | "계단 하행 시 원심성 조절 저하로 불안정, 감독 필요" |
| 보행 속도가 현저히 느림 | 전반적 하지 근력 + 균형 자신감 | "보행 속도 저하, 안전 위해 감독하 보행 권장" |
| 보조도구에 과도하게 의존 | 하지 근력·균형 전반 | "Walker에 상지 과의존, 하지 loading 회피" |

---

## 인수인계 문장 조합 공식

**어느 시기(phase)에 → 무슨 현상이 → 어떤 근육 때문에 → 그래서 어느 정도 도움이 필요한지**

> 예) "Loading response에서 knee buckling 관찰, quadriceps 약화로 보행 시 Min A 및 supervision 필요"

---

## 도움 수준 약어 (참고)

| 약어 | 의미 |
|---|---|
| Indep | 독립 (independent) |
| Sup | 감독 (supervision) |
| SBA | 근접 감독 (stand-by assist) |
| Min A | 최소 도움 (25% 미만) |
| Mod A | 중등도 도움 (25–50%) |
| Max A | 최대 도움 (50–75%) |
| Dep | 전적 의존 (dependent) |

## MMT 등급 (참고)

| 등급 | 소견 |
|---|---|
| G (5) | 최대 저항에 완전 ROM |
| F (4) | 중등도 저항에 완전 ROM |
| F (3) | 중력 이기고 완전 ROM |
| P (2) | 중력 제거 시 완전 ROM |
| T (1) | 근수축만 촉지, 관절 움직임 없음 |
| Z (0) | 수축 없음 |

*"우측 족배굴곡 약함"보다 "우측 tibialis anterior MMT G2"가 인수인계에서 훨씬 명확하다.*
